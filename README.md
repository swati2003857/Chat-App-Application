# 💬 Chat App (MERN Stack)

A real-time chat application built using the MERN Stack with JWT Authentication and Socket.io for instant messaging.

---

## 🚀 Features

- 🔐 User Authentication (Signup/Login)
- 🔑 JWT Authentication
- 💬 Real-time Messaging with Socket.io
- 😊 Emoji Support
- 🔒 Protected Routes
- 📱 Responsive UI
- ⚡ Fast React + Vite Frontend
- 🌐 REST APIs with Express.js
- 🗄️ MongoDB Database

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- DaisyUI
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Socket.io
- Bcrypt.js
- Cookie Parser

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/swati2003857/Chat-App-Application.git
```

Move into the project folder

```bash
cd Chat-App-Application
```

---

## Install Backend Dependencies

```bash
cd backend
npm install
```

---

## Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

---

## Environment Variables

Create a `.env` file inside the backend folder.

Example:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

NODE_ENV=development
```

---

## ▶️ Run Backend

```bash
cd backend
npm start
```

or

```bash
npm run dev
```

---

## ▶️ Run Frontend

```bash
cd frontend
npm run dev
```
